# Balatro 游戏规则手册（LLM Agent 参考文档）

> 本文档描述 Balatro 的客观游戏机制。仅包含游戏规则、数值、交互逻辑，不包含策略建议。

---

## 一、游戏目标

通过 8 个赌注层（Ante 1-8）。每层有 3 个盲注。在每个盲注中，用有限的出牌次数达到目标分数。全部 8 层通关即胜利。

---

## 二、游戏流程

### 2.1 阶段循环

```
盲注选择(PreBlind) → 出牌阶段(Blind) → 结算(PostBlind) → 商店(Shop) → 下一个盲注
```

每个 Ante 包含 3 个盲注：
- **Small Blind**：目标分数最低，可跳过
- **Big Blind**：目标分数中等，可跳过
- **Boss Blind**：目标分数最高，**不能跳过**，附带特殊效果

### 2.2 出牌阶段

1. 从牌堆抽牌填满手牌（默认 8 张）
2. 每回合有固定的**出牌次数**（默认 4）和**弃牌次数**（默认 3）
3. **出牌（Play）**：选 1-5 张牌打出，计算得分，消耗 1 次出牌次数
4. **弃牌（Discard）**：选 1-5 张牌弃掉，从牌堆抽等量新牌，消耗 1 次弃牌次数。弃牌不产生得分。
5. 累计得分达到目标 → 过关
6. 出牌次数 = 0 且弃牌次数 = 0 且得分未达标 → 失败

### 2.3 关键机制

- 出牌次数 = 0 时，无法再打出手牌。此时弃牌不会改变结局（弃完也无法出牌）。
- 分数只增不减（除特定 Boss 效果）。
- 每次出牌后，从牌堆补充手牌至上限。
- 牌堆耗尽时不再补牌。

## 三、计分系统

### 3.1 公式

```
最终得分 = (基础筹码 + 计分牌筹码之和) × (基础倍率 + 加法倍率) × 乘法倍率(X Mult)
```

### 3.2 手牌类型

| 牌型 | 最少张数 | 基础筹码 | 基础倍率 | 条件 |
|------|---------|---------|---------|------|
| High Card | 1 | 5 | 1 | 任意牌 |
| Pair | 2 | 10 | 2 | 两张同点数 |
| Two Pair | 4 | 20 | 2 | 两组对子 |
| Three of a Kind | 3 | 30 | 3 | 三张同点数 |
| Straight | 5 | 30 | 4 | 五张连续点数 |
| Flush | 5 | 35 | 4 | 五张同花色 |
| Full House | 5 | 40 | 4 | 三条 + 对子 |
| Four of a Kind | 4 | 60 | 7 | 四张同点数 |
| Straight Flush | 5 | 100 | 8 | 五张同花且连续 |

- A 在顺子中可作 1（A-2-3-4-5）或 14（10-J-Q-K-A）
- 可以打出少于 5 张牌（如只打 2 张凑 Pair）

### 3.3 计分牌

打出的牌中，只有参与组成牌型的牌计分：
- **Pair/Two Pair/Three of Kind/Four of Kind/Full House**：组成牌型的牌计分，多余的牌不计分
- **Straight/Flush/Straight Flush**：全部 5 张计分
- **High Card**：仅最高一张计分

### 3.4 牌面筹码值

| 点数 | 筹码 |
|------|------|
| 2-10 | 等于点数 |
| J, Q, K | 10 |
| A | 11 |

### 3.5 手牌等级

每种牌型有等级（默认 Lv1）。使用 Planet 卡升级：
- 每升 1 级：基础筹码 +10，基础倍率 +1

## 四、小丑（Joker）

### 4.1 规则

- 最多持有 5 个小丑（Voucher 可增加上限）
- 小丑效果从左到右依次触发
- 小丑在商店购买获得，卖出价 = 买入价 / 2（向下取整）

### 4.2 效果分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 加法倍率(+Mult) | 每次计分时增加固定倍率 | Joker: +4 Mult |
| 乘法倍率(X Mult) | 每次计分时倍率相乘 | Cavendish: X3 Mult |
| 加法筹码(+Chips) | 每次计分时增加固定筹码 | Banner: 每剩余弃牌次数 +30 Chips |
| 花色条件 | 打出特定花色时触发 | Greedy Joker: 每张方块 +3 Mult |
| 牌型条件 | 打出特定牌型时触发 | Jolly Joker: 打出 Pair 时 +8 Mult |
| 计分牌条件 | 对每张计分牌触发 | Fibonacci: 计分牌为 A/2/3/5/8 时 +8 Mult |
| 经济型 | 回合结束产生金钱 | Golden Joker: 回合结束 +$4 |
| 缩放型 | 效果随游戏进程变化 | Green Joker: 每次出牌 +1 Mult，弃牌 -1 Mult |
| 重触发 | 使特定牌多次计分 | Sock and Buskin: 人头牌重触发 1 次 |
| 持有型 | 手中未打出的牌触发 | Baron: 手中每张 K 给 X1.5 Mult |

### 4.3 触发顺序

1. Boss Blind 效果先触发
2. "出牌时"小丑触发（按槽位从左到右）
3. 每张计分牌依次计分（从左到右）：
   - 牌面筹码 → 增强效果 → 版本效果
   - 每张牌触发后，所有"计分时"小丑按槽位从左到右触发
   - 如果有重触发，重复上述过程
4. "持有型"小丑对未打出的手牌触发
5. "独立型"小丑触发

## 六、商店

### 6.1 商店内容

每次过关后进入商店，可以看到：
- 2 个小丑（可重掷更换）
- 1-2 个消耗品（Tarot / Planet / Spectral）
- 1 个凭证（Voucher）— 永久升级
- 1-2 个补充包（Booster Pack）

### 6.2 经济系统

- **利息**：每持有 $5 在回合结束时获得 $1 利息，上限 $5（即持有 $25+ 时每回合固定 +$5）
- **过关奖励**：Small Blind $3，Big Blind $4，Boss Blind $5
- **卖出**：小丑/消耗品卖出价 = 买入价 / 2（向下取整）
- **重掷**：初始 $5，每次 +$1，进入新商店时重置

### 6.3 凭证（Voucher）— 永久效果

| 凭证 | 效果 |
|------|------|
| Grabber | +1 出牌次数 |
| Wasteful | +1 弃牌次数 |
| Crystal Ball | +1 消耗品槽位 |
| Antimatter | +1 小丑槽位 |
| Nacho Tong | +1 手牌上限 |
| Paint Brush | +1 手牌上限 |
| Clearance Sale | 商店物品 75 折 |
| Restock | 重掷费用 -$2 |
| Seed Money | 利息上限 $5 → $25 |

### 6.4 消耗品

| 类型 | 说明 | 来源 |
|------|------|------|
| **Tarot** | 改造牌组（改花色、加增强、销毁牌等） | 商店、Arcana Pack |
| **Planet** | 升级手牌类型等级（+筹码/+倍率） | 商店、Celestial Pack |
| **Spectral** | 强力效果（复制小丑、加版本、销毁+创建牌） | Spectral Pack |

消耗品槽位默认 2 个。

### 6.5 补充包（Booster Pack）

| 包类型 | 内容 | 选择数 |
|--------|------|--------|
| Arcana Pack | 3 张 Tarot | 选 1 |
| Celestial Pack | 3 张 Planet | 选 1 |
| Spectral Pack | 2 张 Spectral | 选 1 |
| Standard Pack | 3 张游戏牌 | 选 1 |
| Buffoon Pack | 2 个小丑 | 选 1 |

## 八、动作空间

| 动作 | 说明 | 合法条件 |
|------|------|---------|
| `select_card_0..7` | 选择/取消选择手牌（切换） | 出牌阶段，对应位置有牌 |
| `play` | 打出已选的牌 | 出牌阶段，plays > 0，至少选了 1 张 |
| `discard` | 弃掉已选的牌 | 出牌阶段，discards > 0，至少选了 1 张 |
| `select_blind_0` | 进入当前盲注 | 盲注选择阶段 |
| `skip_blind` | 跳过当前盲注 | 盲注选择阶段，非 Boss Blind |
| `cashout` | 结算奖励 | 过关后的结算阶段 |
| `buy_shop_item_N` | 购买商店小丑 | 商店阶段，钱够，小丑槽位未满 |
| `buy_consumable_N` | 购买消耗品 | 商店阶段，钱够，消耗品槽位未满 |
| `use_consumable_N` | 使用消耗品 | 持有消耗品时 |
| `sell_joker_N` | 卖掉小丑 | 持有小丑时 |
| `sell_consumable_N` | 卖掉消耗品 | 持有消耗品时 |
| `reroll_shop` | 重掷商店（更换商品） | 商店阶段，钱够重掷费 |
| `next_round` | 离开商店 | 商店阶段 |
| `buy_voucher` | 购买凭证 | 商店阶段，钱够 |
| `buy_pack_N` | 购买补充包 | 商店阶段，钱够 |
| `pick_pack_N` | 选择补充包中的卡 | 补充包打开时 |
| `skip_pack` | 关闭补充包不选 | 补充包打开时 |

### 动作约束

- `play` 只在 plays > 0 且有选中牌时合法
- `discard` 只在 discards > 0 且有选中牌时合法
- plays = 0 且 discards = 0 时，无合法动作，游戏自动判定结果
- `select_card` 是切换操作：选中的牌再次操作会取消选中
- 补充包打开时，只能 `pick_pack` 或 `skip_pack`，其他动作全部锁定
