### B. Missing engine features — **real modeling gaps** — **24 rows** ⚠️
These are the **actual simulator modeling gaps** that need work.

#### B1. Skip-Blind Tag System — **6 rows**
Real: every blind carries a `tag_name` + `tag_effect` drawn from a fixed pool (Negative Tag, Voucher Tag, Coupon Tag, etc.) that activates if the player skips the blind.
Sim: not modeled at all.

Paths: `blinds.small.tag_name`, `blinds.small.tag_effect`, `blinds.big.tag_name`, `blinds.big.tag_effect`, `blinds.boss.tag_name`, `blinds.boss.tag_effect`

Engine work: (1) add Tag type + pool, (2) RNG-roll tag per blind at round start, (3) apply effect on skip.

#### B2. Per-Poker-Hand Statistics — **11 rows**
Real: `hands.*` is a full dict keyed by hand name ("Pair", "Flush", …) with `{level, played, played_this_round, order, chips, mult, example}` per entry. Used for both UI (hand list on right) and scoring (level → chips/mult).
Sim: has `hand_levels: {name: level}` (bare map). Missing `played`, `played_this_round`, `order`, `chips`, `mult`, `example`.

Paths: `hands.High Card` … `hands.Flush Five` (11 hand types including the 3 Steamodded-unlocked types: Flush House, Five of a Kind, Flush Five).

Engine work: extend `HandLevel` struct to carry play counters + base scoring + example cards. Reset `played_this_round` at round boot.

#### B3. Deck Composition Metadata — **2 rows**
Real: `cards.limit = 52`, `cards.highlighted_limit = 5`. Real client always exposes the full current deck composition (52 cards by default, more with Abandoned Deck etc.) alongside limits.
Sim: `cards.limit = None`, `cards.highlighted_limit = None`.

Engine work: add `deck_limit` and `play_limit` (i.e., max cards per play = 5) to snapshot.

#### B4. Pack Limits — **2 rows**
Real: `packs.limit = 2`, `packs.highlighted_limit = 1`.
Sim: `None` / `None`.

Engine work: trivial metadata add.

#### B5. `cards.cards` always populated — **2 rows (value_mismatch)**
Real: `cards.cards` is a list of 52 card dicts (full deck composition) regardless of phase.
Sim: `deck` field is empty (`list[0]`) in Stage_PreBlind; only populated during active round.

Engine work: materialize remaining-deck as `cards.cards` at all times, not just during round.

#### B6. Default Reroll Cost — **1 row (value_mismatch)**
Real: `round.reroll_cost = 5` in shop.
Sim: `shop_reroll_cost = 0` in fresh state (initialized only on shop entry).

Engine work: initialize reroll_cost to 5, not 0.

### C. Deck-modifier application — **~4 rows**

Engine work: plumb `deck` choice into Engine constructor; apply each deck's modifiers (RED: +1 discard, YELLOW: +$10 start, etc.).
